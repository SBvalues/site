About the SBV

The sbv was created in order to avoid scams and make sure new players can trade successfully and be set to be able to understand hit effect values, demand and rarities. This site is mainly used as a value list for the game called "Slap Battles", on roblox. We show values which are usually decided by the community but mostly on their rairty, if an item is limited and it's rarity. This site is an original, which comes with a discord server with hundreds of members creating a friendly community. The discord server includes a "WFL" channel where you can notify trade helpers if you need help with a trade so that you can be extra sure!

Why trust us?

The sbv is trusted by many, being supported by multiple high traders who help create and suggest value changes so that values display accurately and give you the proper information in order to make percise decisions. It has roughly 10k+ views and has hit around 1k total users. If you are ever unsure about a trade simply join the SBV discord server and use the wfl channel!

FAQ |

Frequently Asked Questions

To use the SBV trade calculator simply click "Add" on the left side in order to add that item to your trade, and click "Add" on the right side to add the item onto their side. After adding the items to both sides the trade calculator will display if your trade is Win/Fair/Lose. A win meaning it is a good trade, fair meaning it is fair, while lose means you are losing value.
To join a giveaway you must first join the sbv discord server! From there visit the #giveaways channel where you can participate in weekly giveaways!
Every Friday and sometimes earlier values are updated usually being changed based off poll answers! We show each hit effect value change and exactly what changed, if it went up, or down. This message is in the sbv discord in #website-updates! These messages also contain any addition website information such as any changes to design that were made!
To suggest a value change simply join the sbv discord and find your way to the #website-suggestions channel! There you can create a post and suggest what hit effect value should be changed and why. If we're unsure to accept your acception we usually decide via poll in the #polls channel!(You may also suggest changes via email - sb-values@outlook.com)
